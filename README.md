# AI Fundamentals SCORM Package (Version 1.0)

## AI Fundamentals

This SCORM 1.2 package provides an interactive introduction to AI and Copilot using very basic Python for beginning students.



### Technical Details

**Files Included:**
- index.html - Main lesson with slides
- styles.css - Professional styling with animations
- scorm_api_wrapper.js - SCORM 1.2 API communication
- lesson.js - Navigation, quizzes, and scoring
- imsmanifest.xml - SCORM manifest
- metadata.xml - Course metadata
- images/ - public domain l images

**Passing Score:** 70% 

**WCAG 2 Compliance:**
- Keyboard navigation support
- ARIA labels and roles
- High contrast mode support
- Screen reader friendly
- Focus indicators
- Proper heading hierarchy
