# GitHub Basics SCORM Package (Version 2.0)

## Introduction to GitHub and Version Control

This SCORM 1.2 package provides an interactive introduction to GitHub and version control for programming students.

### Learning Objectives

By the end of this lesson, students will be able to:
- Explain what GitHub is and why it's the industry standard.
- Describe the purpose and benefits of version control.
- Understand what repositories are and how to use them.
- Differentiate between branching and forking.
- Share repositories with instructors and collaborators.
- Apply best practices for using GitHub.
